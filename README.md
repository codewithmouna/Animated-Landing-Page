# Lumina - Animated Landing Page

A modern, responsive landing page featuring smooth animations, scroll reveals, and contemporary design patterns. Built with vanilla HTML, CSS, and JavaScript.

## Features

✨ **Smooth Animations**

- Typing text animation in the hero section
- Floating shape animations
- Scroll-triggered reveal animations for cards

🎨 **Modern Design**

- Dark theme with vibrant accent colors
- Gradient backgrounds and glassmorphism effects
- Fully responsive design

🔧 **Interactive Elements**

- Smooth scroll navigation between sections
- Active link highlighting in navbar
- Hover effects on buttons and cards

📱 **Responsive Layout**

- Mobile-friendly navigation
- Adaptive grid layouts
- Touch-friendly interactive elements

## Project Structure

```
├── index.html          # Main HTML structure
├── styles.css          # All styling and animations
├── script.js           # JavaScript interactions
└── README.md           # Project documentation
```

## Sections

1. **Navigation Bar** - Fixed navbar with smooth scrolling links
2. **Hero Section** - Eye-catching headline with CTA buttons and animated shapes
3. **Features Section** - Showcase of key features with icons
4. **Stats Section** - Display of impressive metrics
5. **Testimonials Section** - Customer feedback carousel

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS variables, gradients, and animations
- **Vanilla JavaScript** - No dependencies required

## Key Features in Code

### Smooth Scrolling

Smooth scroll behavior for all anchor links throughout the page.

### Scroll Reveal Animation

Cards and elements animate into view as you scroll down the page with a 150px threshold.

### Active Navigation Links

Navbar links highlight automatically based on which section is currently in view.

### CSS Variables

Centralized color scheme and spacing using CSS custom properties for easy theme customization.
